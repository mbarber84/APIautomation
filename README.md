# APIautomation

**Using IntelliJ to Automate APIs**

This project is an API automation framework built in **Java** using **IntelliJ IDEA**.  
It contains code and configurations to automate REST API interactions — ideal for validating, testing, and integrating APIs in automated workflows.

---

---

##Features

- Java-based API automation using automated HTTP calls
- Configurable via `config.properties` for easy environment switching
- Build and dependency management with **Maven**
- Structured for expanding test cases and API interaction logic

---

##Prerequisites

Make sure you have the following installed before running this project locally:

- Java **JDK 11+**
- **Maven**
- **IntelliJ IDEA** (or your preferred Java IDE)
